# 04. Folder Structure

## Structure
```text
config/
src/
  api/
  web/
  mobile/
  data/
  framework/
reports/
logs/
docs/
.github/
```

## Purpose
- `config/` holds environment and Playwright settings
- `src/` contains automation code for all domains
- `reports/` stores HTML, JUnit, and Allure output
- `logs/` stores runtime logs
- `docs/` preserves planning and architecture artifacts
